
欢迎来到2025年秋季学期 CS2612《程序语言与编译原理》课程复习仓库！

## 为什么本仓库会出现？
这门课的授课老师（下称“曹大师”）人很厉害，理论水平和编程水平都极高，但他上课的内容我实在是听不懂，可能是我没认真听、听不进去或者实在就是我太笨。

我不知道这门课到底要教什么，在上课的时候我完全没有梳理出这门课的逻辑是什么。看名字，我知道，“编译原理”大概需要教一些和编译器相关的内容，而“程序语言”，则是一些和程序形式化证明相关的内容。即便如此，我上课的时候还是感觉一团乱麻。而老师上课的讲义犹如《春秋》，微言大义，许多地方我压根看不懂，甚至不敢去看，所以我课后也没复习，作业都是糊弄过去的。

在某次Coq（这是一种奇特的编程语言）小测之后，我彻底道心破碎，自此再也没听过课，直接摆烂到期末。

对我来说，《程序语言与编译原理》这门课，太难学了，真的太难学了……

但是，我还是试图弄明白一点点老师上课的内容，用我自己也能听懂的语言，尝试解释一下。

## 本仓库适合谁？
不想看回放，又看不懂讲义，但是希望对课程内容有一些直观理解，以有助于复习的同学。

## 本仓库的内容？
本仓库包含：
- 思维导图（NotebookLM生成）
- 曹大师讲义+理论作业（pdf文件）
- 对讲义的摘录、理解与分析（md文件，包含AI生成内容）
- 两份参考试卷

本仓库不包含：
- 和Coq语言相关的分析、证明（因为期末不涉及）

## 为什么不直接问AI？
我自己在复习时的体会是，AI生成的内容，尽管极为详细，但读起来太累，有时候又缺少对整体逻辑的梳理，甚至不明白我不懂的点在哪。所以，我想还是需要一份包含人工内容的讲义。
